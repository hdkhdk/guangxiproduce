# 八桂星眸 CIS 企业形象识别系统

## 项目概述

这是一个展示广西八桂星眸智能科技有限公司 CIS 企业形象识别系统的单页面网站。

## 技术栈

- HTML5
- CSS3 (现代布局：Flexbox、Grid)
- Vanilla JavaScript
- 无框架依赖，轻量化实现

## 文件结构

```
/新建文件夹
├── index.html      # 主页面（包含所有内容）
└── static/
    └── images/     # 图片资源目录
```

## 页面结构

1. **前言** - 项目背景与战略意义
2. **理念识别系统(MI)** - 企业使命、愿景、价值观等
3. **行为识别系统(BI)** - 内部/外部行为规范
4. **视觉识别系统(VI)** - 基础/应用部分
5. **CIS实施计划** - 执行步骤与评估机制
6. **产品介绍** - 达蒙AI教育机器人、瓦力情感陪护机器人

## 特色功能

- 滚动式单页面导航
- 平滑滚动动画
- 响应式设计（支持移动端）
- 现代科技感视觉风格
- 深蓝色+金色品牌色彩

## 使用方式

直接在浏览器中打开 `index.html` 即可查看。
